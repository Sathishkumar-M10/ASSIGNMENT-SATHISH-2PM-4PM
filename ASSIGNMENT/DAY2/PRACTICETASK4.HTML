HTML

Key points
HTML is a markup language, not a programming language.
It defines the structure of a webpage.
HTML works together with
CSS (for styling)
JavaScript (for interactivity)

CSS

CSS (Cascading Style Sheets) is the styling language used to control the appearance, layout, and design of HTML webpages.
It tells the browser how elements should look, such as:
colors
layout (flexbox, grid)
borders
backgrounds
animations

JS

JavaScript adds behavior to a webpage, such as animations, form validations, pop-ups, interactive buttons, sliders, and more.
What JavaScript Does
Changes HTML/CSS dynamically
Responds to user actions (click, hover, type)
Validates forms
Creates animations and effects
Fetches data from servers (API calls)
Powers web apps (like YouTube, Facebook, Gmail)
